# PickBazar Documentation

# GraphQL 실행
cross-env NODE_ENV=production node dist/server.shop.js

# 플랫폼 실행
yarn server

# Admin Page 실행
node Package로 통합이 안되어 따로 실행 필요
Admin React Page = 3010 Port
Admin GraphQL = 4010 Port

실행 명령(Admin 디렉토리에서 진행) : yarn admin
